  • How to start

  • Requirements:

  • 1) Use: Spigot or Paper

    Without one of these jars the plugin will not work well

  • 2) (optional) use PlaceholdersAPI

    It is optional if you want to use this plugin or not but it can help you with all the plugin messages that work with its api to have custom placeholders from other plugins

  • How to create one server:

    1) install spigot or paper
    2) put that file in a folder
    3) create the archive 'start.bat' with java -Xmx2024M -Xms2024M -jar spigot.jar nogui
    4) open the 'start.bat' and accept the eula
    5) put the plugin in the folder plugins
    6) and you open it again 'start.bat and ready :D
